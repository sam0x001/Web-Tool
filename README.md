# WEB-TOOL
A simple python tool for scanning web applications.

## Installation & Usage
### `python3` is required to run this tool
1- Installtion:
```
pip3 install -r requirements.txt
```
2- Run:
```
python3 web-tool.py
```
Enjoy using it :)

## Options
- Ping Tool
- Server Whois
- Admin page finder
- IP Lookup
- Reverse IP Lookup
- Website analyzer
- IP Information
- Subdomain finder
- Port scanner
- Show HTTP headers
- DNS Resolver
